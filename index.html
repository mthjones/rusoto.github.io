Overview

Rusoto

Rusoto is an AWS SDK written in Rust.

Rusoto consists of two main crates, rusoto and codegen. The codegen crate programmatically generates the source for the API using the AWS API definitions provided by the botocore project. This generated code is then used to construct the rusoto crate, which provides the low-level SDK functionality. Rusoto also provides a helpers crate which provides higher-level abstractions for some of the AWS services (e.g. S3).
