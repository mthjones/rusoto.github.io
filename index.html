Overview

Rusoto

Rusoto is an AWS SDK written in Rust.

Rusoto consists of two main crates, rusoto and codegen. The codegen crate programmatically generates the source for the API using the AWS API definitions provided by the botocore. This generated code in then used to construct the rusoto crate, which provides the low-level SDK functionality. Rusoto also provides a helpers crate which provides higher-level abstractions for some of the AWS services (e.g. S3).

extern crate rusoto;

use std::default::Default;

use rusoto::{DefaultCredentialsProvider, Region};
use rusoto::dynamodb::{DynamoDbClient, ListTablesInput};

fn main() {
  let provider = DefaultCredentialsProvider::new().unwrap();
  let client = DynamoDbClient::new(provider, Region::UsEast1);
  let list_tables_input: ListTablesInput = Default::default();

  match client.list_tables(&list_tables_input) {
    Ok(output) => {
      match output.table_names {
        Some(table_name_list) => {
          println!("Tables in database:");

          for table_name in table_name_list {
            println!("{}", table_name);
          }
        }
        None => println!("No tables in database!"),
      }
    }
    Err(error) => {
      println!("Error: {:?}", error);
    }
  }
}
